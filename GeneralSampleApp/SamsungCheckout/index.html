Samsung Checkout Sample

This sample describes how you can implement billing system on your application. Please click each buttons.


At the first time, call "initialization".
Please note that this sample application is only implemented to be able to test "DUMMY" server.
If you want to use "DEV" or "PRD" server, please replace your information issued by DPI Portal in this sample. (appId, securityKey)
